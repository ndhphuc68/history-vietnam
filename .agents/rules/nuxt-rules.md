# Nuxt Coding Rules

Follow Nuxt 3 architecture.

Project structure must follow:

src/
components/
pages/
layouts/
composables/
stores/
assets/
content/

Rules:

- Pages must be inside pages/
- Components must be inside components/
- Global layout must be inside layouts/
- State must use Pinia
- Shared logic must use composables
