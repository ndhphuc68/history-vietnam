# Tech Stack Rules

This project must use the following technology stack:

Framework:

- Nuxt 3

Language:

- TypeScript

Styling:

- TailwindCSS

State Management:

- Pinia

Rules:

- Always use Nuxt 3 conventions
- Always use <script setup lang="ts">
- Use Tailwind for all styling
- Do not use CSS modules
- Do not use Vuex
- Use Pinia for global state
