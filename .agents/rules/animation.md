# Animation Rules

Animations must be smooth and playful.

Use simple animations.

Do not use heavy animations.

---

# Slide Transition

Slides should transition horizontally.

Animation:

slide-in-right
slide-out-left

Duration:

300ms

---

# Button Animation

Buttons must have:

scale up on hover

scale 1.05

Click animation:

scale down to 0.95

---

# Quiz Feedback

Correct answer:

green highlight
small bounce animation

Wrong answer:

shake animation

---

# Reward Animation

Use celebration animation.

Example:

confetti
star particles

Duration:

1 second
