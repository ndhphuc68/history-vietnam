# Lesson UI Rules

Lessons must use a fullscreen layout.

Structure:

Header
Content area
Navigation button

The content must be centered.

Example layout:

## Header (title)

Story card

---

Next button

Spacing must be large for children.

Use rounded UI.

---

# Story Slide

Story slides display:

illustration
text

Text must be short.

Maximum 2 sentences per slide.

Example:

Card
Image
Text

---

# Quiz Slide

Quiz slides display:

question
3 options

Options must be large buttons.

Buttons must have:

rounded corners
hover animation
click feedback

---

# Navigation

Navigation button:

Next

The button must be large and centered.

Children should easily tap it.

---

# Reward Screen

When a lesson is completed:

Display celebration animation.

Show message:

"Bạn đã hoàn thành bài học!"

Show unlocked hero or badge.
